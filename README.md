Passo a passo de como executar na sua máquina:

- Clone o repositório e acesse a pasta:
``` git clone https://github.com/LucianErick/podcastr-next.git && cd podcastr-next ```

- Instale as dependências:
``` yarn ```

- Inicie a Fake-API:
``` yarn server ```

- Inicie a aplicação:
``` yarn start ```
